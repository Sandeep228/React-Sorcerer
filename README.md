## Prerequisites

Node version 18.7.x

## Cloning the repository

```shell
git clone https://github.com/akshat-g-07/Dukaan-Assignment.git
```

## Install packages

```shell
npm i
```

## Start the dev server

```shell
npm run dev
```

## Build the page

```shell
npm run build
```
